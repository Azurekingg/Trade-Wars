Trading Buddy Videos Directory
==============================

Place your video files here for the Trading Buddy feature.

Supported formats: MP4, WebM, OGG

Video naming convention:
- win_*.mp4 - Videos shown after wins
- loss_*.mp4 - Videos shown after losses
- general_*.mp4 - General trading tips

Example files:
- win_risk_management.mp4
- loss_analysis.mp4
- general_market_trends.mp4
