Video Thumbnails Directory
==========================

Place thumbnail images for videos here.

Requirements:
- Format: PNG or JPG
- Size: 320x180 pixels (16:9 aspect ratio)
- Recommended: PNG for transparency support

Naming convention:
- Match video filename: win_risk_management.mp4 -> win_risk_management_thumb.png
- Or use video ID: win_tip_1 -> win_tip_1_thumb.png
