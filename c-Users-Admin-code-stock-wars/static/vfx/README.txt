VFX Directory
=============

Place your VFX configuration files here.

VFX can be:
- Particle effects (JSON)
- Animation sequences (JSON)
- CSS animations (defined in JSON)
- Shader effects (GLSL/JSON)

Example VFX files:
- profit_glow.json - Glowing effect for profits
- warning_pulse.json - Warning pulse animation
- market_crash.json - Crash effect animation
